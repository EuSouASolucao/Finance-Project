# Finance Project

Sistema de gestão financeira desenvolvido para controle de receitas, despesas, fluxo de caixa e acompanhamento operacional de negócios.

A plataforma centraliza informações financeiras em um único ambiente, reduzindo o uso de planilhas e permitindo uma análise mais precisa e em tempo real.

---

## Problema

Pequenos e médios negócios frequentemente utilizam planilhas para controle financeiro, o que gera limitações como falta de escalabilidade, erros manuais e baixa visibilidade dos dados.

---

## Solução

Este sistema resolve esse problema ao centralizar toda a gestão financeira em uma plataforma web, permitindo organização, automação de processos e análise de dados em tempo real.

## Funcionalidades

- Controle de receitas e despesas
- Fluxo de caixa diário
- Gestão de vendedores
- Controle de comissões
- Dashboard financeiro com indicadores
- Relatórios por período
- Categorização de transações
- Gestão de metas
- Sistema multiusuário
- Controle de acesso por usuário
- Operação em tempo real

---

## Tecnologias Utilizadas

Frontend:
- React
- Vite ou Next.js
- TailwindCSS
- Axios

Backend:
- Node.js
- Express

Banco de Dados:
- PostgreSQL

Outros:
- JWT (autenticação)
- Docker (se aplicável)
